# YouTube Subtitles Viewer Extension

A browser extension that automatically fetches and displays YouTube subtitles for easier viewing.

## Features

* **Easy Subtitle Access**: Quickly view subtitles for any YouTube video that has them.
* **Simple Interface**: A clean and simple interface integrated into the YouTube page.
* **Caption Search**: Allows to look up specific works.

## Installation

To install this extension for development:

1. Clone this repository to your local machine.
2. Open your Firefox browser and navigate to `about:debugging`.
3. Click on "**This Firefox**".
4. Click on "**Load Temporary Add-on...**".
5. Select the `manifest.json` file in the root of this project directory.

The extension should now be installed and active.
